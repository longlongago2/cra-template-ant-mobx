# cra-template-antd-mobx

This is the custom template (Integrates Ant Design and Mobx) for [Create React App](https://github.com/facebook/create-react-app).

For more information, please refer to:

- [Getting Started](https://create-react-app.dev/docs/getting-started) – How to create a new app.
- [User Guide](https://create-react-app.dev) – How to develop apps bootstrapped with Create React App
- [MobX with React](https://mobx-react.js.org) – MobX in React brings easy reactivity to your components. It can handle whole application state as well.
- [Ant Design](https://ant.design) – React is used to encapsulate a library of components which embody our design language. 

## Usage

`npx create-react-app [my-app-name] --template antd-mobx`
